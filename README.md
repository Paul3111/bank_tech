2. How to run the app
   a. If Ruby is not already installed:
   rvm get stable
   rvm use ruby --latest --install --default

   b. Install bundler and rspec
   gem install bundler
   bundle init
   bundle add rspec
   rspec --init

   c. Run Rspec with the following command
   rspec

3. Dependencies
